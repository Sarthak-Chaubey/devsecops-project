📌 Project Overview

This project automates software deployment using Jenkins CI/CD pipeline. Whenever code is pushed to GitHub, Jenkins automatically builds Docker image, removes old containers, and deploys updated application.

📊 CI/CD Workflow

1️⃣ GitHub Integration

Source code is stored in GitHub repository. Developers push updated code to GitHub.

2️⃣ Jenkins Automation

Jenkins fetches latest code and executes pipeline automatically.

3️⃣ Docker Deployment

Docker creates isolated containers and deploys updated application.
