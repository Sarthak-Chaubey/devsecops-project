📌 Project Overview

This project automates the complete software deployment lifecycle. Whenever code is updated on GitHub, Jenkins automatically triggers the pipeline, builds a Docker image, removes old containers, and deploys the updated application.

⚙ Technologies Used

  • Docker Containerization
  • Jenkins CI/CD
  • GitHub Repository
  • Node.js Backend
  • HTML/CSS Frontend
  • Git Version Control

📊 CI/CD Workflow Explanation

1️⃣ GitHub Integration

Source code is stored in GitHub repository. Developers push updated code to GitHub.

2️⃣ Jenkins Automation

Jenkins fetches the latest source code and executes automated pipeline stages defined inside Jenkinsfile.

3️⃣ Docker Build & Deployment

Docker creates an isolated container environment for the application and deploys updated containers automatically.

4️⃣ Continuous Deployment

Old containers are removed automatically and updated applications are deployed without manual work.
