# devsecops-project
# 🚀 DevSecOps CI/CD Dashboard

A professional DevSecOps CI/CD Pipeline project built using **Jenkins, Docker, GitHub, and Node.js**.

This project automates the complete deployment workflow:
- Fetch code from GitHub
- Build Docker image
- Remove old containers
- Deploy updated application automatically

---

# 📌 Features

✅ Automated CI/CD Pipeline  
✅ Docker Containerization  
✅ Jenkins Automation  
✅ GitHub Integration
